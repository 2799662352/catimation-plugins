# Storyboard Light Reconstruction / 光影重构

核心信念:光影不是装饰词,而是画面的物理结构。AI 经常分不清左右、忽略侧光、继承原图旧光影,是因为它更擅长像素概率,不擅长真实三维空间。不要只写“from left / side lighting / cinematic lighting”,要写**光源 + 物理效果 + 阴影结果 + 氛围**。

这个 skill 提炼自中文教程《AI光影重构大师课:从提示词逻辑到后期精准控制》。参考: <https://www.super-i.cn/info-2771.html>

## 先理解 AI 的光影盲区

- **像素逻辑大于空间逻辑**:AI 学到的是“某类光影分布和词汇的相关性”,不是完整三维反射规律。
- **原图残留会干扰**:图生图/局部重绘时,原图已有强光方向会成为惯性,提示词很难直接扭转。
- **左右词不稳定**:只写“左侧光/右侧光”容易漂移。更稳的是写灯位术语和可见效果。

## 提示词公式

```text
光源类型/灯位 + 打到哪里 + 产生什么阴影/高光 + 背景如何暗下去 + 情绪氛围
```

弱写法:

```text
从左侧打光,电影感。
```

强写法:

```text
伦勃朗式侧上方主光只照亮人物右半张脸和手背,鼻梁在另一侧投下三角形阴影,背景保持深暗低照度,皮肤为哑光漫反射,高光柔和不油腻,画面形成高反差但不过曝的戏剧感。
```

## 控光词要写“效果”,不是只写“来源”

优先写这些可见结果:

- 哪一侧脸被照亮,哪一侧进入阴影
- 鼻影、颧骨阴影、下颌阴影如何落
- 背景是否压暗,主体是否从暗背景里浮出来
- 皮肤是哑光漫反射还是油亮反光
- 边缘是否有轮廓光
- 阴影是硬边还是柔边
- 冷暖关系是“暖脸冷背景”还是“冷环境暖实践光”

避免只写:

- cinematic lighting
- beautiful light
- light from left
- dramatic shadows
- high quality light

## 局部光源:拒绝大平光

AI 默认容易把整张脸照亮,导致油腻、塑料、无骨相。用局部光源切开明暗:

```text
复古台灯是唯一主光源,只照亮女孩的侧脸、手指和桌面一小块区域;背景书架落入深蓝绿色阴影,前景暖光与背景冷光形成冷暖对比,皮肤保持 matte skin texture,没有镜面塑料高光。
```

## 去光再加光

复杂图像要二次控光时,不要直接覆盖旧光。先让画面回到均匀散射光,再重建光源:

1. **去光**:让原图恢复低对比、均匀、无明显方向性阴影的散射光状态。
2. **标方向**:用箭头/标注说明新主光和辅光方向。
3. **加光**:按箭头方向增加主光、柔光、轮廓光。
4. **校验**:看阴影方向、皮肤高光、背景明暗是否一致。

示例:

```text
蓝色箭头方向打来伦勃朗主光,红色箭头方向补一盏极弱柔光;主光制造鼻梁侧影和下颌阴影,补光只保留面部暗部细节,不要抹平整体明暗对比。
```

## 什么时候用工具而不是死磕提示词

- **Tapnow 打光工具**:适合新手快速拖拽光源、调颜色和半径;适合单主体、简单场景。
- **NanoBananaPro / GPT Image 二次重构**:适合复杂遮挡、需要先去光再加光、局部重绘、精确箭头控光。

提示词无法稳定扭转原图强光时,直接进入“去光 → 标箭头 → 加光”的后期重构流程。

## 出稿前检查

- 是否只写了“侧光/电影光”,没有写具体物理效果?
- 光照是否只打在必要部位,而不是全局大平光?
- 阴影方向是否跟主光一致?
- 皮肤是否避免油亮塑料高光?
- 是否有冷暖层次或明暗层次拉开空间?
- 图生图是否被原图旧光影干扰?如果是,是否需要先去光再加光?
