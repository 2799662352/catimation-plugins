# Storyboard Live Character Realism / 角色活人感

核心信念:AI 人物“一眼假”往往不是模型不强,而是缺少三块真实感基石:**微神态与潜动作、环境对抗与物理惯性、局部光影与哑光质感**。皮肤和脸已经很真时,真正的假来自“状态的假”。

这个 skill 提炼自中文教程《赋予AI角色“活人感”的3个核心提示词法则》。参考: <https://www.super-i.cn/info-2793.html>

## 三大心法

1. **演戏要给潜台词**:不要写“思考/难过/直视镜头”,要写眼神转移、面部肌肉微动、下意识动作。
2. **动作要给对抗力**:不要让角色在真空里走秀。加风、阻力、重量、环境干扰,让动作有物理原因。
3. **光影要会藏**:不要大平光。用局部光源、冷暖对比、哑光皮肤去掉塑料感。

## 第一招:微表情 + 潜动作

弱写法:

```text
一个穿亚麻西装的男人坐在复古沙发上读信,思考,电影级灯光。
```

强写法:

```text
男人坐在复古沙发上读信,眼神先精准聚焦在信纸最后一行,眉毛微蹙;读到某句话时,视线短暂跳动像在重读,眼睑轻微眨动,嘴唇从轻闭变成微张,随后又因压抑而合上。托着下巴的手不是静止支架,食指和拇指在下颌线缓慢摩挲,带动皮肤产生细微位移,亚麻袖口形成自然褶皱。
```

规则:

- 写情绪的**发生过程**,不要只写结果。
- 静态动作要改成有潜台词的微动作:托下巴 → 摩挲下颌;看信 → 视线跳动/重读/抬眼。
- 手部动作要和心理状态有关:摩挲嘴角、捏信纸、指尖敲杯沿、攥衣角。

## 第二招:环境对抗 + 物理惯性

人物如果像走秀,通常是因为没有外力。给角色一个需要回应的环境因素。

可用环境对抗:

- 风:按住帽子、防止裙摆掀起、头发向一侧飘。
- 雨:擦眼镜、缩肩、避开积水。
- 雪:呼出白雾、脚步变沉、鞋底踩出痕迹。
- 重物:重心偏移、肩膀下压、手臂用力。
- 人群:侧身穿过、被肩膀擦过、短暂停顿。

示例:

```text
在步伐迈出一半时抓拍。一阵微风吹过,她下意识抬起手臂按住太阳帽防风;身体重心微微前倾,轻薄裙摆同时受到风和前进步伐影响向后扬起,头部迎风轻轻倾斜。
```

这比“女人在花园散步”更真实,因为动作有了原因。

## 第三招:局部光源 + 冷暖对比 + 哑光皮肤

塑料感常来自大平光:整张脸都亮,没有阴影过渡,高光油腻。解决方式:

- 用局部光源,只照亮侧脸/手/桌面一小块。
- 背景压暗,让主体从阴影里浮出来。
- 前暖后冷或冷环境暖实践光,拉开空间层次。
- 强制 `matte skin texture` / 哑光皮肤,避免塑料高光。

示例:

```text
复古台灯是唯一主光源,只照亮女孩的侧脸、双手和键盘区域;背景书架落入幽暗蓝绿色冷光,前景暖光和后景冷光形成对比。皮肤为哑光漫反射,没有油亮高光,阴影柔和地雕刻鼻梁和下颌线。
```

## 图生视频:底图定生死,视频提示词做减法

如果首帧已经锁定构图、长相、光影和质感,视频 prompt 不要再重复“冷暖对比、台灯、哑光皮肤、电影级”等词。只写最小动作:

```text
她轻微呼吸,胸口有很小的起伏;手指在键盘上轻敲两次,眼神仍停在屏幕上,台灯阴影自然附着在脸和手上。
```

## 出稿前检查

- 人物是否有潜台词,而不是静态摆姿势?
- 是否写出情绪流动:起始 → 过渡 → 高潮/收住?
- 手部动作是否有心理或任务原因?
- 是否存在环境对抗力:风/雨/雪/重量/阻力?
- 光影是否避免大平光和塑料高光?
- 图生视频时是否删掉重复风格词,只驱动微动作?
