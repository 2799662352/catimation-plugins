# Storyboard Style Extraction Logic / 风格即选择

核心信念:风格不是元素堆砌,而是一套反复执行的选择逻辑。不要问“图里有什么”,要问“它拒绝了什么”“多张图共同保留了什么”“哪些约束一旦去掉就不再是这个风格”。

提炼自中文教程《风格即选择:AI画面风格提取三步法》。参考: <https://www.super-i.cn/info-2586.html>

## 第一阶段:从“看有什么”到“看没什么”

普通反推会得到语义拟合:

```text
女人、隧道、奔跑、模糊、灯光。
```

这只能复刻内容,不能复刻风格。

正确提问:

```text
请不要描述这张图里有什么。请分析这张图拒绝了什么:
1. 它拒绝了哪些颜色?
2. 它拒绝了哪种清晰度/焦点?
3. 它拒绝了哪种构图秩序?
4. 它拒绝了哪种光线?
5. 哪些条件一旦去掉,这张图就不再成立?
```

得到的是不可变条件,例如:

- 低快门拖影/抽帧,拒绝清晰冻结。
- 高密度青绿色调,拒绝暖色和纯白高光。
- 幽闭隧道透视,拒绝开阔空间。

## 第二阶段:样本提纯,找共性不是找个例

一张图容易过拟合。至少准备 3 张同风格但内容不同的图:

- 样本 A:人物行走
- 样本 B:冲突/动作
- 样本 C:亲密/静态

共性提取指令:

```text
请比较这三张参考图。忽略其中偶然的主体、道具和剧情,只提取跨样本稳定出现的视觉约束:
1. 色彩范围和禁止色
2. 光源方向与明暗关系
3. 空气介质和环境质感
4. 构图尺度与人物占比
5. 镜头焦段和透视结构
6. 负向约束
请输出一份可复用 Style DNA。
```

## 第三阶段:四维风格说明书

把风格标准化为 4 个板块:

### 1. 空间与构图

```text
IMAX/70mm 极宽画幅,人物占比极小,环境压迫主体,水平线稳定,强调宏大尺度。
```

### 2. 光线与氛围

```text
自然动机光源,空气中有尘埃/雾气/霜雪,光线有可见介质,拒绝影棚柔光。
```

### 3. 色彩与质感

```text
低饱和、哑光、胶片颗粒、风化旧感。地球端偏 dusty beige, 太空端偏 sterile grey / deep black。
```

### 4. 限制与规则

```text
禁止 neon、cyberpunk、glossy plastic、clean digital art、vibrant colors、holograms。
```

```text
Style DNA:

6. 色彩与质感:
7. 负向约束:
8. 可迁移提示词:
```

## 出稿前检查

- 是否只提取了主体/道具,没有提取选择逻辑?
- 是否分析了“没什么”和“拒绝什么”?
- 是否使用多张参考图避免过拟合?
- 是否输出负向约束?
- 是否能把风格迁移到完全不同的主体上?
