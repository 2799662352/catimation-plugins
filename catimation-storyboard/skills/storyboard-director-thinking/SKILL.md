# Storyboard Director Thinking / AI 视频导演思维

核心信念:工具进步不等于创作能力进步。真正决定作品上限的是导演思维:空间调度、核心行动、剪辑与补拍。AI 直出只是素材,导演负责节奏和意义。

提炼自中文教程《告别工具人:3招掌握AI视频导演思维》。参考: <https://www.super-i.cn/info-2655.html>

## 方法一:调度优先,不是分镜优先

场面调度是“导演如何安排镜头框里的一切”。提示词必须包含:

### 1. Z 轴纵深

```text
前景: 虚化的百叶窗边缘遮挡画面左侧
中景: 男人坐在桌边,身体一半落入阴影
后景: 书架和走廊尽头的冷色窗光
```

### 2. 动机光源

不要写“昏暗光线”,要写光从哪里来:

```text
桌面复古台灯是唯一暖色主光,只照亮人物半张脸和手;左侧百叶窗透入冷色月光,在墙面切出横向阴影。
```

### 3. 视差运镜

不要只写 zoom in。用空间产生运动:

```text
镜头从百叶窗后方缓慢向右平移,前景百叶窗快速掠过,中景人物移动较慢,后景书架产生真实视差。
```

## 方法二:叙事优先,不是画面优先

一场戏必须围绕一个核心行动展开。

弱写法:

```text
女主非常悲伤,眼泪流下,唯美电影光影。
```

强写法:

```text
女主在雨中疯狂翻找垃圾桶,试图找回被丢掉的戒指;手指被铁皮划破但没有停下,雨水和血水混在一起,她只盯着垃圾袋深处。
```

规则:

- 把形容词降级,把动词升级。
- 情绪要通过目标、阻力和行动表达。
- 角色必须在“做一件事”,不是站着展示状态。

## 方法三:要有剪辑与补拍思维

不要迷信一次生成 15 秒完整戏。真正电影感来自切点、插入、空镜和节奏突变。

流程:

1. **A-roll 主镜头**:让模型跑完整基础动作。
2. **找情绪断点**:例如角色看向窗外的瞬间。
3. **补拍特写**:瞳孔极特写、手指松开、杯子掉落。
4. **补拍空镜**:窗外飞船阴影、手机落地、门把手轻响。
5. **剪辑组合**:主镜头打底 + 特写突刺 + 空镜留白。

示例:

```text
A-roll: 男人走进咖啡厅,点咖啡,坐下,看向窗外。
B-roll 1: 惊恐瞳孔极特写,窗外冷光在虹膜中闪过。
B-roll 2: 咖啡杯从手中滑落,慢动作砸在地上。
B-roll 3: 窗外巨大飞船阴影扫过街道。
```

## 导演思维 Prompt 模板

```text
核心行动: [角色正在做什么,目标是什么]
环境阻力: [什么阻碍他/她]
空间调度:
- 前景:
- 中景:
- 后景:
动机光源:
- 主光:
- 辅光/环境光:
运镜:
- 机位:
- 运动:
- 视差:
剪辑计划:
- A-roll:
- B-roll:
- 情绪断点:
```

## 出稿前检查

- 是否先设计空间调度,而不是只生成漂亮分镜?
- 是否有前景/中景/后景?
- 光源是否有动机?
- 角色是否有核心行动和阻力?
- 是否把 AI 直出当素材,并计划补拍/剪辑?
