# Storyboard Character Acting / AI 视频角色演技

核心信念:高级演技不是把“开心、愤怒、悲伤”写在脸上,而是让观众从**微表情的过渡、身体的下意识反应、环境声音线索**里自己读出情绪。AI 角色像 NPC,通常不是因为脸不够精致,而是因为情绪被写成了标签,动作没有因果,环境像静音壁纸。

这个 skill 提炼自中文教程《告别“NPC感”:三步重塑AI视频角色演技》的核心方法。参考: <https://www.super-i.cn/info-2797.html>

## 三步演技公式

写角色演技 prompt 时,按这个公式组织:

```text
基础景别与主体 + 面部微表情序列(眼/唇/肌肉) + 协同肢体小动作 + 环境物理动态(声音线索) + 摄影与光影风格
```

不要写:

```text
她很开心 / 他很生气 / 她极度悲伤 / 角色紧张地看向镜头
```

要写:

```text
她先低头垂眸,眼睑轻微颤动两次,呼吸变浅,嘴唇微张但没有立刻笑;随后缓慢抬头,嘴角只上扬一点,手指仍攥着衣角,像是在努力把局促藏起来。
```

## 第一步:用微表情替代情绪标签

AI 容易把情绪词映射到模板表情:happy → 标准微笑,angry → 皱眉瞪眼,crying → 流泪。要拆成生理层面的局部变化。

### 紧张 / 恐惧

- 眨眼频率加快
- 瞳孔轻微收缩
- 喉结或喉部吞咽
- 视线短暂闪躲
- 呼吸变浅,胸口起伏变小

示例:

```text
不要写“她很害怕”。写:她的眼睛先快速眨动两次,瞳孔轻微收缩,喉部做出一次克制的吞咽动作,视线从对方脸上滑开0.5秒后又强迫自己看回去。
```

### 强忍 / 压抑

- 眼眶微红但没有落泪
- 下颌骨肌肉瞬间绷紧
- 嘴唇压成一条短线
- 鼻翼轻微扩张
- 目光停住,但身体不爆发

示例:

```text
她没有哭,只是眼眶慢慢泛红;下颌肌肉绷紧约0.4秒,嘴唇压住,手指在袖口内侧攥紧后又松开。
```

### 害羞 / 内敛

- 视线不自觉向右下游离
- 睫毛轻颤
- 嘴唇微张后又闭合
- 肩膀轻微内收
- 抬头动作比正常速度慢

示例:

```text
她先低头垂眸,睫毛轻微颤动,呼吸带动肩膀小幅起伏;嘴唇微张像要说话却停住,随后缓慢抬头,只露出很浅的微笑。
```

## 第二步:用肢体因果建立真实感

情绪不会只停在脸上。它会蔓延到肩膀、手、脖子、胸口和重心。角色如果脸上很焦虑,身体却站得笔直、双手自然垂落,就会产生“NPC感”。

### 肩膀与体态

- 紧张/防备:肩膀微缩,胸口收紧,身体重心后移。
- 无奈/卸力:肩膀下沉,背部松掉,头微微低下。
- 愤怒/克制:胸口起伏变大,肩颈肌肉变硬,动作幅度反而变小。

### 手部下意识安抚动作

人在情绪波动时会做自我安抚行为。用这些动作打破“模特站姿”:

- 搓手
- 攥紧衣角
- 手指敲击桌面一次
- 摸脖子
- 抓耳朵
- 推眼镜
- 指尖轻轻滑过花瓣/杯沿/信纸

示例:

```text
她低头看着手里的野花,没有直接抬头;指尖极慢地沿花瓣边缘滑过,像在感受纹理,随后眼神先向右试探,伴随一次轻微深呼吸,才慢慢抬头望向远处。
```

## 第三步:用环境音线索打破“MV式静音感”

即使目标模型暂时不直接生成音频,也要在画面里放入能“让观众听见”的视觉线索。不要让人物生活在真空玻璃罩里。

### 声音视觉化

用视觉动态暗示声音:

- 窗帘被微风吹动 → 暗示风声
- 老旧挂钟摆动 → 暗示滴答声
- 火炉跳动 + 茶水热气 → 暗示炉火与水汽
- 草叶轻摆 + 浮尘穿过光束 → 暗示山风和安静空气
- 雨滴打在玻璃上 → 暗示压抑空间
- 远处门轴轻动 → 暗示有人靠近

示例:

```text
她没有说话,房间里只有挂钟摆动的暗示:背景墙上的钟摆轻轻来回,窗帘被夜风掀起一角,桌上茶水冒出的热气在侧光里断续上升。
```

## 情绪类型写法

### 局促 / 紧张 / 防备

核心:身体收缩 + 自我安抚。

```text
中近景固定镜头。角色肩膀微缩,身体重心略向后,一只手无意识攥住衣角;眼睛快速眨动两次,视线从对方脸上避开,又在0.6秒后慢慢看回来。
```

### 压抑 / 愤怒 / 隐忍

核心:肌肉极度紧绷 + 小幅物理爆发。

```text
他没有大喊,只是胸口起伏变重;下颌线绷紧,右手指节压在桌面上逐渐发白,杯子被推移不到2cm,随后所有动作突然停住。
```

### 释怀 / 失落 / 卸力

核心:重力感。让紧绷肌肉失去支撑。

```text
她听完那句话后没有立刻哭;肩膀先下沉约1cm,手里的纸慢慢垂到腿边,呼气让胸口松掉,眼神从对方脸上滑向地面。
```

### 生活化破冰小动作

核心:一点不完美的毛边,打破精致假人感。

```text
他说话前先推了一下眼镜,指尖在镜框上停顿;随后像是意识到自己太紧张,轻轻挠了挠耳后,笑容只出现一半就收住。
```

## 实战铁律

1. **做减法,锁死固定镜头**  
   微表情、肢体小动作、环境动态已经足够丰富时,优先使用固定镜头或限制景别,避免大幅推拉摇移导致主体变形。

2. **加慢速,留出发呆空间**  
   高频使用“缓慢地、微妙地、轻柔地、极小幅度”压制 AI 过度表演,让情绪有酝酿时间。

3. **造介质,打破真空感**  
   水蒸气、晨雾、光束、飘叶、浮尘、雨滴、火光都是连接人物与环境的桥梁。

4. **顺逻辑,动作不能打架**  
   写“双手端茶碗”就不要再写“用手整理头发”;写“垂眸深思”就不要同时写“直视镜头微笑”。

5. **稳底座,优先图生视频**  
   用高质量静态首帧先锁构图、长相、光影和质感,视频 prompt 专心驱动微动态,成功率更高。

## 何时调用其他 skill

- 角色动作需要明确动机:调用 `storyboard-character-motivation`。
- 需要把参考视频拆成时间线/空间结构:调用 `storyboard-scene-breakdown`。
- 需要更物理化的动作参数:调用 `storyboard-physics`。
- 需要声音分层设计:调用 `storyboard-audio`。

## 出稿前检查

- 是否还在使用“开心/生气/悲伤/紧张”这类总结词?
- 情绪是否拆成眼、唇、肌肉、呼吸、肩膀、手部动作?
- 脸部变化和肢体动作是否因果一致?
- 是否有至少一个生活化小动作打破摆拍?
- 是否有环境物理动态暗示声音或温度?
- 镜头是否足够稳定,没有让大运镜抢走演技?
- 动作指令之间是否互相打架?
