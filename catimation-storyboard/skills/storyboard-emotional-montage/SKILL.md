# Storyboard Emotional Montage / 情绪蒙太奇

核心信念:情绪不是让角色把“悲伤/愤怒/紧张”写在脸上,而是让导演用环境、对比和特写安排观众的注意力。真正控制情绪的是视听语言,不是表情标签。

提炼自中文教程《打破AI摆拍感:3个情绪蒙太奇技巧》。参考: <https://www.super-i.cn/info-2674.html>

## 技巧一:用环境代替情绪

弱写法:

```text
一个年轻人在房间里非常伤心,哭泣,沮丧。
```

强写法:

```text
广角镜头,一个孤独背影坐在昏暗空荡的公寓里,背对镜头。大雨拍打巨大玻璃窗,水滴模糊窗外城市霓虹。桌上有一盆枯萎植物,冷蓝和灰色光影压住整个房间。
```

规则:

- 把抽象情绪投射到天气、光影、空间和静物。
- 人物可以不露脸,让世界替他表达。
- 环境物要有情绪对应:枯萎植物、湿玻璃、空椅子、破杯子、停摆钟。

## 技巧二:用对比制造情绪冲突

库里肖夫效应:观众对一个镜头的理解,取决于前后镜头。

不要连续堆同一种情绪:

```text
奔跑 → 追车 → 砸桌 → 争吵
```

改成反差:

```text
镜头A: 家中聚会极度热闹,人群拥挤,暖光闪烁,笑声和碰杯动作充满画面。
硬切镜头B: 同一角色独自坐在走廊尽头冷光里,身影很小,手里还握着没喝完的纸杯。
```

规则:

- 极动之后接极静。
- 大喜之后接空落。
- 人群之后接孤单角落。
- 明亮之后接冷暗。

## 技巧三:用特写放大情绪

情绪高潮不要用半身或全身。特写的作用是剥夺观众全局视野,强迫注意力落在局部。

手部特写:

```text
极端手部特写,男人的右手死死攥住一张皱掉的辞职信,指节发白,纸边被汗水浸湿,桌面暖光只照到手背和纸角,背景完全虚化。
```

眼部特写:

```text
极端眼部特写,瞳孔微微放大,下眼睑轻颤,眼眶里只有一层薄薄水光但没有流泪;冷色窗光映在虹膜边缘,背景被压成黑暗。
```

## 镜头组合模板

```text
情绪目标: [孤独/压迫/崩溃/释然]

镜头1 环境客观对应物:
[天气/静物/空间衰败/光影]

镜头2 对比镜头:
[喧闹 vs 安静 / 拥挤 vs 空旷 / 暖光 vs 冷光]

镜头3 极端特写:
[手/眼/嘴角/物件局部] + [微动作] + [光影]

剪辑:
[硬切/延迟切/声音先行/静音留白]
```

## 出稿前检查

- 是否把情绪结果改成环境客观对应物?
- 是否用镜头前后关系制造情绪,而不是单镜头堆情绪?
- 情绪高潮是否使用局部特写?
- 是否有极动/极静或热闹/孤独的反差?
- 是否避免“角色在镜头前表演情绪”的图库感?
