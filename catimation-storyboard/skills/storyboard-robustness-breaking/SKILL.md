# Storyboard Robustness Breaking / 鲁棒性破坏

核心信念:AI 的鲁棒性会把画面推向“最安全、最典型、最稳定”的模板答案:主体居中、光线均匀、背景干净、风格统一。高级感来自可控范围内的轻微失稳。

提炼自中文教程《鲁棒性破坏:掌控AI的“可控失控感”》。参考: <https://www.super-i.cn/info-2464.html>

## 不是乱,是可控偏移

鲁棒性破坏不是让画面崩坏,而是:

```text
能控制方向,但不控制所有细节。
```

目标效果:

- 松弛
- 真实
- 轻微不完美
- 混乱度刚刚好
- 情绪自然
- 没有塑料模板感

## 方法一:构图破坏

让画面不再标准居中:

```text
off-center composition, imperfect framing, slightly tilted horizon, foreground interruption, subject partly blocked by a passing object, candid snapshot feeling, negative space on one side
```

适用:

- 人物更自然
- 时尚街拍
- 纪实摄影
- 生活感场景

## 方法二:光线破坏

破坏 AI 默认的均匀柔光:

```text
uneven natural light, accidental light spill, mixed color temperature, imperfect exposure, light leaking from the edge of frame, reflected light from nearby surfaces, unstable brightness
```

适用:

- 去掉影棚糖水感
- 制造真实现场光
- 胶片/纪实/杂志感

## 方法三:风格破坏

风格太统一会像 AI。加入轻微风格噪音:

```text
subtle texture inconsistency, slight color drift, analog imperfection, uneven film grain, imperfect material rendering, minor edge roughness, handmade visual noise
```

## 组合模板

```text
主体与方向:
[主体/场景/情绪方向]

构图基准 + 构图破坏:
[基本景别] + off-center / foreground interruption / imperfect framing

光线基准 + 光线破坏:
[主光类型] + uneven light / accidental spill / mixed temperature

风格基准 + 风格破坏:
[胶片/写实/杂志] + subtle texture inconsistency / analog imperfection
```

## 使用边界

- 商业产品图:只轻微破坏构图和光线,不要破坏产品形态。
- 人像:可破坏背景、光线、构图,但保留脸部识别。
- 艺术图:可加强风格破坏和材质偏移。
- 连续角色:不要破坏角色锚点,只破坏摄影层。

## 出稿前检查

- 是否仍然是居中、均匀光、干净背景?
- 是否至少有一个轻微不稳定维度?
- 失控是否服务于真实/松弛,而不是降低质量?
- 主体识别和商业目标是否仍然清楚?
- 是否避免提示词过长导致重新变稳定?
