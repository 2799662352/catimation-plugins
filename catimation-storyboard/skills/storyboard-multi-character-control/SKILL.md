# Storyboard Multi Character Control / 多角色控制

核心信念:多角色失败不是因为 AI “不懂人”,而是因为你把多个角色、多个动作、多个时序塞进同一句话,导致注意力偏移和动作稀释。多角色必须用**空间占位、时间段、保留/修改语义编辑、分镜接力**来控制。

提炼自中文教程《90%的人踩坑:AI视频多角色失控?3个技巧拿捏!》。参考: <https://www.super-i.cn/info-2679.html>

## 方法一:空间占位 + 时间段拆分

弱写法:

```text
咖啡厅里,左边男人喝咖啡,右边女人跳舞,接着男人站起来鼓掌。
```

问题:AI 可能让男人一边端杯一边鼓掌,或只保留女人跳舞。

强写法:

```text
空间锁定:
- 左侧三分之一:男人坐在靠窗桌前,右手拿咖啡杯,身体保持坐姿。
- 右侧三分之一:女人站在吧台前,留出完整动作空间。

时间线:
0-2s: 左侧男人只做喝咖啡动作,右侧女人保持静止等待。
2-5s: 右侧女人开始跳舞,左侧男人放下咖啡杯并转头看她。
5-8s: 女人动作减弱,男人站起鼓掌,镜头保持固定。
```

## 方法二:语义编辑要明确“保留”和“修改”

当一个角色满意,只改另一个角色时,不要全局重生成。使用视频编辑/语义编辑并写清楚:

```text
保持右侧女人的外貌、动作、服装、位置、光影和背景完全不变。只修改左侧男人:让他从坐姿变为站起鼓掌,不要改变右侧女人。
```

万能公式:

```text
保留 [角色A 的身份/动作/位置/服装/光影/背景] 不变。
只修改 [角色B] 的 [具体动作/表情/姿态]。
其他人物和环境不得重绘。
```

## 方法三:复杂动作拆成多镜头

不要把“拔剑、冲刺、闪避、跳起、转体、劈砍、落地”塞进一个长镜头。拆成阶段:

```text
镜头1 特写: 女主手指收紧,握住旧车票,眼眶微红,风吹乱发丝。
镜头2 中景跟拍: 她松开车票,转身向男主奔跑,车票从前景飘落。
镜头3 全景慢动作: 两人在站台中央相拥,老式火车从旁边呼啸而过,强风扬起裙摆。
```

多阶段接力时,只需统一:

- 角色外貌锚点
- 服装和核心道具
- 环境描述
- 色彩/光影基调

不必强求每块砖完全一致,景别切换会让观众自动忽略小差异。

## 多角色提示词模板

```text
角色锚点:
- A: [外貌/服装/位置/任务]
- B: [外貌/服装/位置/任务]

空间占位:
- A 位于 [画面区域], 动作空间 [范围]
- B 位于 [画面区域], 动作空间 [范围]

时间线:
0-2s: [谁动,谁不动]
2-5s: [谁反应,谁保持]
5-8s: [动作转移或结束]

禁止:
- 不要互换服装/动作
- 不要让 A 的动作污染 B
- 不要全局重绘已满意角色
```

## 出稿前检查

- 是否明确每个角色的画面位置?
- 是否每个时间段只有一个主动作?
- 是否写清楚“谁保持不变”?
- 是否避免“同时/接着/然后”这类自然语言流水账?
- 复杂动作是否拆成多个镜头接力?
