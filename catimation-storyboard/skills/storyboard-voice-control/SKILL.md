# Storyboard Voice Control / 角色声音控制

核心信念:控制声音不是写“愤怒、悲伤、激动”,而是写角色**怎么把情绪说出来**。真实声音包含年龄感、声线质感、语速、停顿、气息、重音、尾音、发声习惯和心理状态。

这个 skill 提炼自中文教程《如何控制 AI 角色的语气、语速和音色?》。参考: <https://www.super-i.cn/info-2785.html>

## 声音公式

```text
年龄感 + 声线质感 + 语速 + 情绪底色 + 发声习惯
```

弱写法:

```text
年迈谋士严肃地说。
```

强写法:

```text
年迈谋士的声音低沉干哑,语速缓慢,每个短句之间保留半秒停顿;音量不高,但咬字清晰,句尾略微下沉,像在克制地施压。
```

## 音色不要只写身份

身份词只能给 AI 大概方向,不够稳定。把身份拆成声音特征:

- **少年将军**:年轻但压着怒气,语速偏快,尾音短促,咬字硬。
- **年迈谋士**:低沉沙哑,语速慢,停顿多,句尾下沉。
- **冷静女主**:气息稳定,音量不高,字清楚,重音少但很准。
- **暴躁反派**:音量突然抬高,语速快,重音砸在关键词上,尾音短断。
- **疲惫旁白**:气声比例高,语速慢,句间留白,尾音散开。

## 语气:情绪词是结果,不是过程

把情绪转换成声音结构:

### 愤怒

- 音量提高
- 语速加快
- 咬字变重
- 句尾收紧
- 爆发前可以先短暂停顿

```text
前半句保持平静讽刺,语速稳定;说到“不”时突然加重,音量抬高一档,后半句语速加快,尾音压低并短促收住。
```

### 悲伤

- 音量下降
- 语速变慢
- 气息变轻
- 尾音拖长
- 关键字前有犹豫停顿

```text
第一句声音很轻,尾音微微上扬像在等待答案;第二句语速放慢,气息变薄,句尾拖长,最后一个字几乎被呼吸带走。
```

### 冷静

- 语速稳定
- 音量中低
- 停顿精确
- 字清楚
- 情绪不外露

```text
她用稳定的中低音说话,每个字都清楚,没有明显情绪起伏;只在关键名词前停顿0.4秒,让压迫感来自节奏而不是音量。
```

## 多句台词按时间顺序写声音变化

长对话不要一句话概括“越来越生气”。逐句写:

```text
角色声音设定: [年龄感] + [声线质感] + [整体语速] + [情绪底色]
第一句: [初始声音状态] + [语速] + [停顿]
第二句: [情绪变化] + [重音位置] + [句尾处理]
第三句: [音量变化] + [爆发或收住]
第四句: [最终情绪落点]
```

示例:

```text
角色声音:中年父亲,低沉略哑,整体语速偏慢,情绪底色是压抑。
第一句“你先坐下。”声音低,语速慢,句号后停0.6秒。
第二句“这件事,我听他们说过。”逗号处明显停顿,重音落在“他们”。
第三句“但我想听你亲口说。”音量略微提高,尾音压低,没有吼。
第四句“现在。”短句,低音,完全收住。
```

## 标点就是节奏器

- **逗号，** 短暂停顿,语速稍慢。
- **句号。** 完整停顿,语气落点明确。
- **省略号……** 延长停顿,犹豫、思考、未说完。
- **感叹号！** 情绪爆发,语速加快,音量抬高。
- **换行** 段落停顿,适合情绪转折。
- **短句** 节奏加快,适合压迫、命令、爆发。
- **长句** 情绪铺陈,适合解释、犹豫、试探。

## 输出模板

### 单人单句

```text
角色声音设定:
- 年龄感:
- 声线质感:
- 语速:
- 情绪底色:
- 发声习惯:

台词表达方式:
- 音量:
- 停顿:
- 重音:
- 尾音:
- 气息:

台词:
"..."
```

### 多句台词

```text
角色声音设定:
[年龄感] + [声线质感] + [整体语速] + [情绪底色]

声音时间线:
1. "...": [声音状态]
2. "...": [情绪变化]
3. "...": [重音/爆发/收住]
4. "...": [最终落点]
```

## 出稿前检查

- 是否只写了“愤怒/悲伤/激动”,没有写声音过程?
- 是否有年龄感和声线质感?
- 是否写了语速、停顿、气息、重音、尾音?
- 多句台词是否按时间顺序递进?
- 标点是否承担节奏功能?
- 每个角色的声音是否能被听出区别?
