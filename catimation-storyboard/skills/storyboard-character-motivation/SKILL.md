# Storyboard Character Motivation / 人物行为动机

核心信念:人物像真人,不是因为脸更精致,而是因为他/她**有事可做**。不要只写“抬头、回头、微笑、慢慢走路”。要写清楚:人物正在经历什么、动作为什么发生、动作之后身体和环境有什么轻微反应。

这个 skill 提炼自中文 AI 视频提示词教程《让AI人物更像真人的关键,不是脸,而是“有事可做”》的核心方法:人物动作要有处境、动机、连续变化和微弱生命反应。参考: <https://www.super-i.cn/info-2832.html>

## 行为动机四问

写任何人物动作前,先回答这四个问题:

1. **人物在哪里?** 场景要能带出行为约束:等车站台、雨夜街口、宫殿长廊、厨房台面、电梯里、咖啡店角落。
2. **人物现在在做什么?** 写具体任务,不要只写姿势:读信、看手机、转杯子、擦雨水、听门外声音、等人、收拾桌面。
3. **动作为什么此刻发生?** 触发因素可以是脚步声、消息弹窗、风吹乱头发、对方靠近、一句台词、物体滑落、突然安静。
4. **动作之后有什么小反应?** 眼神停顿、呼吸变化、手指用力、重心移动、纸张变皱、衣袖随呼吸起伏、发饰轻晃、嘴唇轻抿。

弱写法:

```text
古风女子站在房间里,手里拿着书信,缓慢看向镜头。
```

强写法:

```text
古风女子刚读到书信里一行意外内容,手指把纸边捏紧约1cm,目光在墨迹上停顿0.6秒,随后缓慢抬向门口;她的袖口随着一次轻呼吸下落,纸张留下细小折痕。
```

## 单人物视频公式

单人物为主时,用这个结构:

```text
人物身份 + 具体场景 + 当前状态/任务 + 动作原因 + 2-3个连续动作 + 微弱生命反应 + 镜头语言
```

逐项检查:

- **人物身份**:一句话即可,如“年轻宫女”“疲惫外卖员”“中年厨师”。
- **具体场景**:地点 + 时间 + 氛围。
- **当前状态/任务**:动作开始前,人物正被什么事情占据。
- **动作原因**:为什么现在要动。
- **连续动作**:2-3个前后相连的小节拍,不要堆无关动作。
- **微弱生命反应**:呼吸、眼神聚焦、手部压力、发丝/衣料/饰品晃动、重心变化。
- **镜头语言**:推近、静态中景、浅景深、过肩、轻微手持漂移。

示例:

```text
夜晚庭院廊下,一位古风女子独自坐在低案旁,盯着未写完的书信出神。远处一声脚步打断她的思绪;她悬在纸上的右手停住,指尖压住桌沿约3mm,眼睛先移向走廊但头没有完全抬起。50mm慢慢推近,浅景深,灯笼光在袖面轻微跳动。
```

## 微弱生命反应

庄重、等待、对峙、出神这类场景不适合大动作。真实感来自“小反应”,不是一直动。每个镜头选 2-4 个即可,太多会像编舞。

可用反应词库:

- 抬眼前先有短暂停顿
- 手指轻轻收紧/松开/敲一下
- 呼吸带动衣领或袖口起伏几毫米
- 发丝、耳坠、流苏、胸牌在动作后轻微归位
- 嘴唇轻抿、下颌肌肉收紧、喉结吞咽
- 重心从一只脚转到另一只脚
- 手停在正在做的任务中,不要“戏剧性僵住”
- 物件跟着反应:纸张压出折痕、杯子转动、手机屏幕反光移动

尽量用物理描述替代情绪标签:

- 用“视线在门口停住0.7秒”,不要只写“她很紧张”。
- 用“拇指压住杯柄直到指节发白”,不要只写“他很焦虑”。
- 用“呼气后肩膀下沉约1cm”,不要只写“她放松了”。

## 多人物:一个人行动,另一个人反应

多人物最容易假,因为模型会让每个人都同时表演。正确做法是: **一个人先行动,另一个人产生反应**。

结构:

```text
角色A先做一个具体动作 → 角色B注意到并产生轻微反应 → 角色A停顿或调整 → 气氛发生变化
```

例子:

- 一个人靠近,另一个人后退半步,先避开视线,再慢慢看回来。
- 一个人说话,听的人手停在物件上,随后指尖收紧。
- 一个人伸手,另一个人身体不动,只把手向后缩回2cm。
- 一个人停下脚步,另一个人先察觉到安静,只转动眼睛而不是立刻转身。

双人镜头里要显式写动作顺序:“A先行动”“B反应”“A停住”。这能防止模型让两个人各自动一套、没有关系。

## 反模式

这些写法会制造“AI摆拍感”:

- 只写外貌:脸、皮肤、衣服、电影光,但没有正在做的事。
- 动作没有原因:缓慢回头、微笑、看镜头,但没有触发因素。
- 过度表演:安静场景里做大动作,头发/衣服一直飘,所有人同时动。
- 雕塑站桩:站在宫殿里,没有呼吸、手部压力、眼神方向或物件互动。
- 情绪只有形容词:悲伤、紧张、感动,但没有身体证据。

## 出稿前检查

交付 prompt 前检查:

- 每个可见动作都有触发原因吗?
- 人物动作开始前,是否正在做一件具体事情?
- 是否有 2-3 个连续节拍,而不是一个孤立动作?
- 微反应是否是身体/物件层面的,必要时可量化?
- 多人物镜头里,是否是一个人行动、另一个人反应?
- 如果删掉所有“脸好看/皮肤细腻/电影感”词语,这个镜头仍然成立吗?如果不成立,补任务、处境和反应。
