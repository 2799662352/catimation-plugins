Structure Rules:
1. Mid-action snapshot: freeze at peak tension, never "then/after"
2. 2-4s per shot, ONE core action
3. Micro-expression: physiological with specific measurements (e.g. brow furrowed Xmm, pupil dilation Xmm — you decide the numbers based on intensity), never emotional labels (happy/sad)

## Example

- ✗ "she gets angry then leaves" (uses "then", two actions, an emotion label).
- ✓ Mid-action freeze, ONE action: "[char1] slams the mug onto the table — peak tension; brow furrowed 4mm, jaw clenched, pupil 3mm; 2.5s."
