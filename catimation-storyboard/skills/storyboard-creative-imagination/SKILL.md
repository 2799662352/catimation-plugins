# Storyboard Creative Imagination / 创意想象力

核心信念:当所有人都能做高画质视频时,真正留住观众的是“违和感”和“观看方式”。不要只追求真实,要敢于重写规则:物理可以反常,时间可以出错,镜头可以成为角色。

提炼自中文教程《别再死磕“大片感”!3个高级思路,重塑你的AI想象力》。参考: <https://www.super-i.cn/info-2671.html>

## 思路一:打破物理定律

用反常理制造停留:

- Anti-gravity / 反重力
- Levitating / Floating subtly / 漂浮或轻微悬空
- Flowing upwards / 水流向上
- Weightless debris / 失重碎片
- Massive object behaving lightly / 巨物轻盈化

示例:

```text
电影级真实摄影。一只毛发逼真的北极熊坐在长满青苔的巨大岩石上刷手机,但岩石悬浮在半空中。背景森林瀑布违背重力向天空倒流,水珠和碎石失重漂浮。整体保持真实摄影质感,不要卡通化。
```

视频续写:

```text
保持画面的失重状态。瀑布持续向上倒流,空中碎石和水珠缓慢漂浮,北极熊稳定悬浮,爪子轻轻滑动手机屏幕。
```

## 思路二:让时间失控

时间创意不是慢放/加速,而是时间发生异常:

- 主体静止,背景高速流逝
- 动作做到一半突然冻结
- 光影快速变化,人物不变
- 过去和现在同框
- 某个局部时间倒流

示例:

```text
城市十字路口的长曝光时间异常画面。周围行人和车流形成高速运动模糊,霓虹拖成光线轨迹;画面中心的女孩完全静止,发丝、衣角、眼神都像被时间冻结。镜头固定,冷色夜光,强烈时空割裂感。
```

动作阻断:

```text
男人抬手准备接住杯子,动作停在半空;杯子碎片和水滴也悬浮在空中,只有窗外雨水仍在正常下落。
```

## 思路三:把镜头变成角色

镜头不是旁观记录工具,可以拥有身份:

- First-person POV from a cup / 从杯子内部看世界
- First-person POV from a stray cat / 流浪猫视角
- Extreme low angle as an insect / 昆虫贴地视角
- Security camera POV / 监控视角
- Inside helmet / 从头盔内部看
- From behind dirty glass / 脏玻璃后偷窥

示例:

```text
第一人称 POV,镜头位于透明水杯内部,透过弯曲玻璃和水面折射看见餐桌另一侧的人。杯壁边缘形成巨大模糊前景,人物脸部被水纹轻微扭曲,桌面灯光在玻璃上产生高光反射。
```

## 创意 Prompt 模板

```text
常规世界: [真实环境/主体]
被打破的规则: [重力/时间/视角/比例/物理常识]
可见证据: [哪些物体表现异常]
保持真实的部分: [光影/材质/摄影质感/环境逻辑]
视频动作: [只驱动异常规则延续,不要让模型纠正回正常]
```

## 出稿前检查

- 画面是否只有“高画质”,没有规则反转?
- 反常理是否有明确可见证据?
- 是否保留真实材质和摄影质感,避免变成卡通胡闹?
- 视频 prompt 是否强调“保持异常规则”,防止模型自动纠正?
- 镜头是否可以从物体/动物/低处/内部视角重写?
