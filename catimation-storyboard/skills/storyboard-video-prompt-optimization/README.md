# storyboard-video-prompt-optimization —— sd2-pe 的衍生模块

> 本 skill 是 **`sd2-pe`(视频与提示词的总兜底/起点)的衍生模块**,在 sd2-pe 产出的结构**之内**做精简与权重再分配,不替代它、不打破其八大要素/三段论骨架。

## 分工

| 层 | skill | 负责 |
|---|---|---|
| 骨架(先) | **sd2-pe** | 工程化提示词的完整结构:八大要素、路径 A/B 三段论、兜底包 |
| 衍生(后) | **storyboard-video-prompt-optimization**(本 skill) | 在该结构内瘦身:尊重首帧霸权、文本主要驱动运动、单一主导逻辑、化解镜头/光影指令冲突、避免动作过多稀释运动权重、消除 PPT 感 |

## 何时启用

当 sd2-pe 产出的提示词出现 **太长 / 风格词堆砌 / 首帧权重被忽视 / 运镜冲突 / 动作敷衍** 等病灶时,用本 skill 在原结构内精简,而非重写。

> 即:**sd2-pe 搭好骨架,本 skill 负责"删到刚刚好",让运动权重不被稀释。**
