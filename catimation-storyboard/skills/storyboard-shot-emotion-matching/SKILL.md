# Storyboard Shot Emotion Matching / 镜头情绪匹配

核心信念:不是所有情绪都适合所有镜头。景别控制心理距离,焦段控制空间压迫,空间逻辑决定真实感。不要堆“广角、特写、长焦、浅景深、背景压缩”,先决定观众应该站在多远、用什么透视感受情绪。

提炼自中文教程《3个维度彻底讲透镜头情绪匹配》。参考: <https://www.super-i.cn/info-2652.html>

## 维度一:用景别控制心理距离

### 远景 / 极远景

用途:渺小、孤独、宿命、环境压迫。

```text
极远景镜头,一个微小独行者穿深色冬装,在巨大冰川雪原上跋涉;远处山脉占据大部分画面,天空阴沉,人物只占画面很小比例。
```

### 中景

用途:叙事基底,交代谁在什么地方做什么事。

```text
中景镜头,腰部以上。年轻女孩靠在火车窗边,白色亚麻衬衫被黄昏风轻轻吹动,窗外灰尘微粒在光束中漂浮。
```

### 特写 / 极特写

用途:情绪高潮武器。不要一直用,关键时刻使用。

```text
极端眼部特写,眼眶有水光但没有流泪,下眼睑轻颤,窗外冷光映在虹膜边缘。
```

## 维度二:用焦段控制空间压迫

### 14-24mm 广角

作用:透视拉扯、边缘畸变、不安、压迫、动荡。

```text
低角度广角街头摄影,16mm 镜头。雨夜湿沥青反射混乱霓虹,前景地面纹理巨大且清晰,行人从镜头边缘模糊掠过。
```

### 35mm

作用:环境叙事、人物和空间平衡。

```text
35mm 镜头,纪实电影剧照。渔民在黄昏潮湿码头修补渔网,环境占主导,前景绳索纹理和远处港口灯光都可见。
```

### 50mm

作用:接近人眼,平稳、真实、不过度戏剧化。

```text
50mm 夜景街头胶片摄影,湿润城市街道反射霓虹和车灯,行人打伞经过,焦点落在湿沥青纹理。
```

### 85-200mm 长焦

作用:空间压缩、隔离、偷窥感、环境窒息。

```text
200mm 长焦镜头,疲惫上班族站在拥挤十字路口,背景建筑被压缩得靠近人物,前景路人肩膀严重虚化遮挡,浅景深孤立他的脸。
```

## 维度三:统一空间逻辑

错误组合:

```text
远景,人物站山顶,极浅景深,广角镜头,背景被拉近压缩。
```

问题:

- 远景通常需要环境信息,不适合极浅景深。
- 广角会拉开纵深,背景压缩是长焦特性。

正确做法:

```text
先选一种光学逻辑:
- 压迫感: 特写/中景 + 广角畸变 或 长焦压缩 + 高对比光影
- 渺小感: 极远景 + 广角/全景 + 深景深 + 环境主导
- 亲密感: 中近景/特写 + 85mm + 浅景深 + 柔光
- 纪实感: 35mm/50mm + 自然光 + 较深景深 + 环境细节
```

## 出稿前检查

- 景别是否符合情绪距离?
- 焦段是否和空间效果一致?
- 是否存在广角与长焦特性互相冲突?
- 是否先确定摄像机位置、镜头焦段、与人物距离?
- 是否用镜头语言表达情绪,而不是只写 8k/电影级?
