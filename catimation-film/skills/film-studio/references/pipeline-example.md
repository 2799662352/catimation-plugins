# 完整样例:从一句概念到成片

走一遍九道门,演示「制片导演」如何编排本地工艺 skill。示例:**《最后一班地铁》**,30s 都市悬疑微电影,实拍质感。

---

## G0 · 概念
- **Logline:** 一个加班到深夜的白领,在末班地铁上发现对面座位坐着「今早出门的自己」。
- 类型:都市悬疑 / 时长:30s / 画幅:16:9 / 受众:短视频悬疑向 / 画风:低饱和胶片质感、实拍感。
- **门 G0:** 用户确认 → 进 G1。

## G1 · 剧本(调 `screenwriter`)
- 三拍微结构:常态(疲惫上车)→ 触发(看见另一个自己)→ 转折(对方抬头微笑)。
- 产出:分场 + 极简对白(几乎无台词,靠画面)。
- **门 G1:** 用户批准 → 进 G2。

## G2 · 分镜镜头表(调 `scene-blueprint` + `director-narrative-flow` + `storyboard-shot-emotion-matching`)

| 镜 | 景别 | 运镜 | 时长 | 动作 | 转场 |
|----|------|------|------|------|------|
| 01 | 远景 | 静止 | 4s | 空荡车厢,主角低头刷手机 | cut |
| 02 | 中景 | 慢推 | 6s | 抬头,视线越过过道 | match cut |
| 03 | 特写 | 静止 | 5s | 对座=另一个自己,同样疲惫 | cut |
| 04 | 近景 | 微晃 | 8s | 主角僵住,对方缓缓抬头 | cut |
| 05 | 特写 | 静止 | 4s | 对方嘴角上扬,车厢灯闪 | 黑场 |
| 06 | 远景 | 拉远 | 3s | 灯灭再亮,对座空了 | end |

- **门 G2:** 用户批准镜头表 → 进 G3。

## G3 · 角色与场景设定(调 `director-anchor-extraction-quality` + `director-character-consistency`,台账用 `animation-craft` continuity-ledger)
- **主角锚点:** 30 岁男、短发、灰西装外套敞开、白衬衫领口松开、黑框眼镜、左手戴旧手表。
- **场景锚点:** 末班地铁车厢、冷白顶灯偏右、窗外漆黑、地面反光、广告灯箱在右后。
- **银幕方向:** 主角朝右(过道在右);轴线固定。
- **门 G3:** 用户批准设定 → 进 G4。

## G4 · 逐镜出图(本 app 图像工具 + `director-cinematic-composition` + `director-lighting-continuity`)
- 每条出图 prompt **逐字粘入** G3 锚点。
- 例(镜 03 首帧):`末班地铁车厢内,冷白顶灯偏右,中央过道反光;对座坐着30岁男、短发、灰西装外套敞开、白衬衫领口松开、黑框眼镜、左手旧手表,疲惫低头;16:9,低饱和胶片质感,前景过道虚化遮挡(见 storyboard-foreground-occlusion)`。
- **门 G4:** 用户逐场挑选关键帧 → 进 G5。

## G5 · 图生视频(调 `seedance-video-craft`,实拍非动画故不叠 animation-craft)
- 模式:图生视频(已有关键帧),Seedance 2.0,720p。
- 镜 04 运动描述:`8s——0–2s 主角僵住屏息(身体先于表情);2–5s 对座的"自己"缓缓抬头(弧线);5–8s 四目相对,车厢灯轻微频闪;镜头近景微晃。声音:地铁运行床 + 抬头瞬间环境压低半拍。禁止漂移:两人脸型/眼镜/手表一致。`
- 生成前爆款体检:hook(0–3s 空车厢+另一个自己)够不够钩?→ 把镜 03 的"揭示"提前。
- **门 G5:** 用户逐镜审片段(用 `animation-craft` 审查 rubric)→ 进 G6。

## G6 · 配音配乐(调 `storyboard-audio` + `storyboard-voice-control`)
- 配乐:低频嗡鸣 + 心跳,张力 6→9;音效:地铁运行、灯管频闪滋滋、末镜灯灭的"啪";几乎无对白。
- **门 G6:** 用户批准 → 进 G7。

## G7 · 拼接成片(调 `ffmpeg-win`)
- 装配表:6 镜按时长拼,镜 02→03 用 match cut,镜 05→06 用黑场过渡;混音;导出 1080p 上采、9:16 另出一版竖屏。
- **门 G7:** 用户审粗剪 → 进 G8。

## G8 · 体检与交付
- 整片体检:hook 强、单一主导(自我对峙)、payoff(对座空了)兑现;分心风险低。
- 交付制片包(见 `production-package.md`)+ 可选叫 `trailer-plan-generator` 出 6s 预告。
