# 交付制片包格式

G8 最终交付。把全流程的产出汇总成一个结构化制片包,方便复盘、返工、续作。

```markdown
## 制片总览
[片名 / 类型 / 目标时长 / 画幅 / 画风 / 场数 / 镜数 / 模型]

## 剧本(G1)
[logline + 分场剧本/对白,或链接到 screenwriter 产出文件]

## 镜头表(G2)
[逐镜:景别 / 运镜 / 时长 / 动作 / 转场;附分镜 JSON]

## 角色与场景设定(G3)
[逐角色:锁定锚点(脸/发/剪影/服装/标志道具)]
[逐场景:光源/天气/建筑锚点]
[连贯性台账(逐镜结束态)]

## 关键帧(G4)
[逐镜首帧/关键帧文件 + 对应出图 prompt(含锚点)]

## 视频片段(G5)
[逐镜片段文件 + Seedance prompt + 模式/时长/分辨率 + 体检分]

## 声音(G6)
[配乐方案 / 音效表 / 配音稿 + 语气参数 / 节拍对齐点]

## 装配时间线(G7)
[装配表:镜号 / 时长 / 转场 / 调速 / 音频层;导出规格(分辨率/画幅/码率)]
[多版本:横屏成片 / 竖屏 9:16 / 预告片]

## 体检与生产笔记(G8)
[整片爆款体检:hook / 留存 / payoff / 分心 评分]
[生成顺序、一致性清单、模型/工具记录、已知问题、续作建议]
```

---

## 交付检查清单

- [ ] 每镜都有「在时间线里的位置」(没有孤立片段)。
- [ ] 角色身份跨全片一致(对照 G3 锚点抽查首/峰/末帧)。
- [ ] 场景光源/天气/银幕方向跨镜连贯。
- [ ] 末镜→下镜的衔接帧能接上(对照 continuity-ledger)。
- [ ] 声音覆盖每个关键节拍,无突兀静音/爆音。
- [ ] 无不想要的文字/字幕残留/logo/水印(除非项目需要)。
- [ ] 导出了所需的所有版本(横/竖/预告)。
- [ ] 整片体检过阈值;不过则定位到具体镜返工(回对应门)。
