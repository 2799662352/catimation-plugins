# マルチカット 連続性管理台帳

マルチカット/シーケンスの着手前にこの表を埋め、1カット生成するごとに「終わりの状態」を更新し、次カットの最初の画の入力とする。目的:同一性ドリフト・場面リセット・小道具消失を防ぐ。

```text
プロジェクト:[名称]    シーケンス:[シーン/段落]    モデル:[Seedance / 即夢 / 可灵 / 海螺]

【グローバルアンカー(全編で不変)】
キャラ同一性:[顔/髪/シルエット/服装の類別/象徴的小道具/ポーズの癖]
場面同一性:[固定建築/光源/天候/地面の痕跡]
画面の向き:[誰が左/右、カメラ軸線]
スタイル:[画風/カラーグレード/質感、director-anime-quality-boost を参照]

【カット別 状態表】
| カット | 主芝居 | 小道具状態 | 身体状態(傷/濡れ/塵/情感) | 光/天候 | 最後の画=終わりの状態(=次カット継承) |
|----|--------|---------|--------------------------|---------|------------------------|
| 01 |        |         |                          |         |                        |
| 02 |        |         |                          |         |                        |
| 03 |        |         |                          |         |                        |
```

## 記入例(雨夜の再会、3カット)

```text
プロジェクト:雨夜の再会    シーケンス:路地口の対峙    モデル:Seedance 2.0

【グローバルアンカー】
キャラ同一性:卵形の輪郭、黒い低い結び髪、左頬の小さなほくろ、ベージュのトレンチ、傘を低く垂らす癖。
場面同一性:濡れた路地、ネオンのピンク+ブルー二色光源が右上、降り続く小雨、水たまりの反射。
画面の向き:少女は右を向く、相手は画面右側。カメラ軸線は固定。
スタイル:日本式セルルック cel、低彩度のフィルム感、厚塗り禁止(director-anime-quality-boost 参照)。

【カット別 状態表】
| カット | 主芝居 | 小道具状態 | 身体状態 | 光/天候 | 最後の画=終わりの状態(=次カット継承) |
|----|--------|---------|----------|---------|------------------------|
| 01 | 背を向けて立つ、肩が沈む | 赤い傘を低く垂らし滴る | 濡れ髪はまだ貼らず、平静 | ネオン右上、小雨 | 振り向きの途中、傘が上がり始める |
| 02 | 振り向き+傘上げ、ゆっくり寄り | 赤い傘が斜め上へ | 瞳孔がわずかに縮み、呼吸が一瞬止まる | 同上、雨音が半拍下がる | 正面の寄り、傘は斜め上、濡れ髪が頬に貼る |
| 03 | 一歩近づく | 赤い傘が前傾し顔を半分隠す | 濡れ髪が頬に貼り、目に光を含む | 同上、雨脚がやや増す | 二人が同一フレームに入る、傘の縁が触れ合う |
```
