# 絵コンテ コマ雛形

各コマが6つの問いに答える:前コマとの変化 / まずどこを見るか / キャラの芝居 / シルエットは読めるか / カメラ距離 / どれだけの尺か。

```text
PANEL [番号]
サムネ/文字記述:[簡略画か一文]
サイズ(景別):ロング / ミドル / アップ / インサート / 移動カット
前コマとの変化:
観客の視線が最初に落ちる点:
キャラの芝居/決定:
シルエットの可読性:可 / 要簡素化
カメラ距離 + カメラワーク:
尺の見積り:[秒]
次コマへの転換:カット / ディゾルブ / マッチカット / 時間跳躍
音の手がかり:
```

---

## 記入例(アニメ短編の一節:3コマ)

```text
PANEL 01
サムネ:猫が机の縁にうずくまり、縁へ転がるグラスを見つめる。
サイズ:ミドル
前コマとの変化:シーケンス開始、常態を確立。
観客の視線が最初に落ちる点:グラス(高コントラスト・縁寄り)。
キャラの芝居/決定:猫が前脚を一本伸ばし、探る(予備動作)。
シルエットの可読性:可。
カメラ距離 + カメラワーク:ミドル、フィックス。
尺の見積り:2s
次コマへの転換:カット
音の手がかり:グラス底と机面のわずかな擦れ音。

PANEL 02
サムネ:前脚が軽く押し、グラスが机の縁を越える。猫の目が見開く。
サイズ:アップ(前脚+グラス)
前コマとの変化:グラスが落下を始める(転換)。
観客の視線が最初に落ちる点:接触点。
キャラの芝居/決定:押し出した直後に前脚を引っ込める(中断)。
シルエットの可読性:可。
カメラ距離 + カメラワーク:アップ、わずかに落下をフォロー。
尺の見積り:1s
次コマへの転換:マッチカット(落下に沿って)
音の手がかり:接触音+その後半拍の沈黙(間)。

PANEL 03
サムネ:グラスが床で砕ける。猫が机の縁から顔を覗かせ、耳を後ろに伏せる。
サイズ:ロング
前コマとの変化:帰結の着地+反応(締め)。
観客の視線が最初に落ちる点:破片、次に猫の顔。
キャラの芝居/決定:顔を覗かせ、耳を伏せる(反応/余韻)。
シルエットの可読性:可。
カメラ距離 + カメラワーク:ロング、フィックスで画面を呼吸させる。
尺の見積り:3s
次コマへの転換:—(段落終了)
音の手がかり:破砕音+環境の尾音。
```
