# ビジュアルストーリーテリング

アニメ短編・アニメシーン・絵コンテ・アニマティック・ゲームの一場面・AI動画ショットが、まず画と芝居で伝え、説明は後にすべきとき読み込む。

## 素材参照とオリジナリティ(学習は実素材を必ず引用)

ここで示すのは汎用のアニメ工芸。ただし**学習・分析の段階では、実在の作画素材・アニメーター情報・作品を必ず引用して学ぶ**——これは欠かせない。権威ソース(Sakugabooru / Sakuga Blog / Sakuga Journal / Animétudes / ANN / AniDB)でカット・担当者・分析を当たり、リンク・作者名・作品名・話数を明記する。商業アニメも学習参考として分析・リンク列挙してよい。詳細な採取フローは `references/animator-technique-reference.md`。

学習で得たものを**成果物(生成 prompt / 完成映像)に落とすとき**だけ、一線を引く:商業カットを1カット丸ごと逐写ししない、他者のオリジナルキャラを自作の成果として流用しない、固有名の画風指定(「〇〇監督風で生成」)を避ける。抽出した着想は中立な技法語に翻訳する:「読めるステージング」「明快なシルエット」「小道具駆動の因果」「抑制されたタイミング」。

## 視覚ビート点検

台詞・エフェクト・磨きの前に、このビートが**見て分かる**ことを確認する:
- **最初の画:** 観客が被写体・場所・現在の圧力を識別できる。
- **欲求:** ポーズ・視線・手の位置・移動方向が「何をしたいか」を示す。
- **障害:** 小道具・床面・距離・閾・規則・別キャラが芝居を阻む。
- **変化:** カットが芝居の後に「何が違うか」を見せる。
- **帰結:** 最後の画が次ビートを継ぎやすくする。

観客が基本の芝居を理解するのに一段落の説明が要るなら、ステージングを簡素化する。

## 無声の芝居

無声、または主に視覚で見せる場面は、まず身体の言語、次に顔の細部:
- **方向:** 前傾・振り向き・手を伸ばす・後退・停止が意図を示す。
- **重量:** 立ち姿・バランス・沈み・持ち上げ・引きずり・反発が力みを示す。
- **間:** 一つのホールドポーズが気づき・気まずさ・驚き・決定を示す。
- **中断:** 完了の前で止める方が、完遂より表現的なことが多い。
- **収まり:** 衝突後にキャラがどう落ち着くかが、笑いや情感を担う。

顔はビートを鋭くできるが、身体と小道具の関係が芝居を担うべき。

## 場面は制約である

設定が「何が起こり得るか」を変えるとき、初めて意味を持つ:
- 狭い道は葛藤や気まずいタイミングを作る。
- 滑る床はバランスを変える。
- 高所は恐れ・スケール・地位を変える。
- 扉・壁・橋・机・道具は規則を作る。
- 何もない空間は小さな芝居を露わにする。

場面が物語を語る意図なら、背景を装飾にしない。場所に「押し返させる」。

## リズムと呼吸の余地

速い芝居はコントラストがあるとき読みやすい:
- 強い芝居の前に短いセットアップ。
- 衝突に明快な接触のコマを。
- 不意の後に短い反応をホールド。
- 切り替える前に最後の画を呼吸させる。
- 観客が主芝居を追えないなら、余分な運動を削る。

全要素を常時動かさなくても、場面はエネルギーで満ちる。

## 音の手がかり

最終素材が無音でも、音をリズムの一部として設計する:
- 芝居の前の小さな音が注意を促す。
- 接触音は衝突に合わせる。
- 不意の後の沈黙がビートを着地させる。
- 反復音がコメディのリズムや機械的圧力を作る。
- 尾音が次カットへ橋渡しする。

音は画がすでに伝えたものを鋭くするためにある。

## ガードレール(学習は奨励、成果物で一線)

著名な作品・アニメーターを参照するとき:
- **学習・分析は積極的に。** 実在の作品・作者・カットを当たり、出典リンク・作者名・作品名・話数を明記して、何の表演ロジックを抽出するかを具体化する。
- **成果物に落とすときだけ一線:** 象徴的なポーズ・正確な段取り・台詞・構図・キャラ設計の丸写しは避け、原理を保って状況・キャラ・ステージング・締めを変える。
- 生成指示に固有名の画風(「〇〇監督風で生成」)を直接書かない。代わりに「明快に劇的なシルエット」「小道具駆動の無声の段取り」「手描き線のエネルギー」「コメディの反転前の短い劇的ホールド」のような技法語で書く。

目標は、実素材から学んで移植可能な工芸へ翻訳すること。学習段階の引用は欠かせない。
