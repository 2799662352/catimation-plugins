# ショット運動の経験則10

カット/絵コンテ/アニマティック/AI動画 prompt に実務の運動判断が要るとき読み込む。

## 法則1:芝居は一つが先導する

各カットには主導する芝居/変化が一つ要る。

- 主体を最初に動かす/変える。
- 副次運動は、そのビートを支えない限り静かに。
- カメラ・キャラ・天候・粒子・布・光を全力で競わせない。

観客が「何が起きたか」を言えないなら、運動が多すぎるか、間違ったものを動かしている。

## 法則2:最初の画は読めること

最初の画が答えるべきこと:
- 誰/何を見ているか。
- どこにいるか。
- すでにどんな圧力/期待があるか。
- 視線はどこへ向かうべきか。

綺麗だが不明瞭な最初の画=そのカットの無駄。

## 法則3:止め(ホールド)はビートである

止めが芝居を読ませる。
- 重要な決定の前にホールド。
- 不意・衝突・情感の転換の後に間を取る。
- 切り替える前に最後の画を呼吸させる。
- 沈黙や減衰した音で注意を鋭くする。

毎秒に新運動を詰めない。

## 法則4:運動方向は意味を運ぶ

画面の向きとカメラ方向はビートを支える。
- カメラへ向かう → 脅威・親密・到達。
- カメラから離れる → 喪失・逃走・スケール・余韻。
- 左右の向きは連続カット間で一貫させる。
- 垂直運動 → 落下・上昇・圧力・解放・権力・脆さ。

カット間で方向を反転しない。反転自体が強調される物語ビートの場合を除く。

## 法則5:状態は変わる、同一性は変わらない

連続性は「綺麗さ」ではなく「同一性+帰結」。
- 保つ:顔/髪・シルエット・服装の類別・象徴的小道具・場面アンカー。
- 変える:身体状態・残留・損傷・情感・小道具状態・天候・光(物語が要るとき)。
- マルチカットでは明記:次カットが継承する状態。

## 法則6:カメラは可読性に奉仕する

カメラワークは芝居の代用ではない。
- 芝居や変形を明快に見せたいとき:フィックス(止め)。
- TU(トラックアップ/寄り)→ 気づき・圧力。
- 引き → 帰結・スケール。
- フォロー → 空間を横切ることに意味があるとき。
- 提示(パン/ティルトの開示)→ 隠れた情報自体が要点のときだけ。

カメラが要(かなめ)の接触/衝突/表情を隠すなら、簡素化する。

## 法則7:衝突には接触と余韻が要る

強い芝居には読める連鎖が要る:
1. 予備動作。
2. 接触または変化。
3. 反応。
4. 残留または新状態。

余韻が無いと、芝居に重みが出ない。

## 法則8:ループには変化か目的が要る

ループが向くもの:呼吸・待機・環境の生気・儀式・機械・コメディの反復。
- ループを生かすには小さな変化を足す。
- 物語情報が別にあるときはループを静かに。
- キャラが決定する/場面が転じるときはループを破る。

## 法則9:環境は応答する

世界が応えるとき、アニメ映像は地に足が着く:
- 足が接地すると塵が舞う。
- 布が身体運動に追従する。
- 炎が動くと光がちらつく。
- 衝突で水が波紋を打つ。
- 群衆・葉・煙・影が遅延した時序で応える。

環境反応で、力・スケール・帰結を証明する。

## 法則10:ピークのコマを検証する

多くの映像はカット中盤でだけ破綻する。
- 最初の画・ピークの芝居・衝突のコマ・最後の画を確認する。
- 探すもの:顔の崩れ・手の歪み・小道具の消失・カメラの遮蔽・場面リセット・フリッカー。
- 最後の画は、次カットへ橋渡しできる使える状態であること。
