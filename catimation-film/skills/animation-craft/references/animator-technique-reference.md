# 動画人 技法リファレンス(技法参考 / 模倣ではない)

## 大方針(最重要)

学ぶのは **「表演ロジック(芝居・タイミング・間・コマ・エフェクトの設計思想)」**。そして学習段階では **実在の素材・作者・作品を必ず引用して参照する**(下記の権威ソース)。

- ✅ **学習・分析段階:** 実在の作画素材・アニメーター情報・作品を**積極的に引用**する。リンク・サムネ情報・作者名・担当カットを明記して、何から何を学ぶかを具体化する。商業アニメも学習参考として分析対象にし、リンクを列挙してよい。
- ✅ **抽出するもの:** タイミングの配分、タメ・ツメ、間の取り方、芝居の機微、エフェクトの設計思想、コマ打ちの判断。
- ⚠️ **成果物(生成 prompt / 完成映像)に落とすとき:** 抽出した**技法・表演ロジックを技法語で**書く。商業カットを1カット丸ごと逐写ししたり、他者のオリジナルキャラを自作の成果として流用したりはしない。固有名の画風指定(「〇〇監督風で生成」)も避ける。
- これが**より安全**(著作権・オリジナリティ)であり、かつ**より効く**——モデルは「誰々風」より「タイミングと芝居の具体」を渡した方が狙った表演を掴む。
- ✅ **アニメも実写も:** 実在の映画撮影・照明・カラーの技法ドキュメントも検索し、**実在の技法用語**(焦点距離・絞り・照明・色温度・カラーグレードなど)で prompt を埋める。
- ✅ **正向で書く:** 「〜しない」(反向プロンプト)を並べず、望む状態を肯定形の技法語で指定する。

各項目は **「学ぶ表演ロジック」→「AI映像 prompt への落とし方(技法語で)」** の順で読む。

**このスキルが起動し、作画リファレンスが要るときは、内部知識だけで済ませず、まず能動的にウェブを検索・取得する。** 以下のサイトは欠かせない。学習・分析でリンクとクレジットを明記して引用する。具体的なアニメーター名・作品名・技法を当たるとき、不確かなまま書かず、必ず一次情報源に当たる。

|---|---|---|

| **Sakuga Blog** | https://blog.sakugabooru.com/ | 作画の分析記事(誰が・どの回・どのカットを・どう作画したか)。 |
| **Sakuga Journal** | https://sakugajournal.wordpress.com/ | 演出・芝居の批評(例:山田尚子の芝居設計)。 |

| **Anime News Network(ANN)** | https://www.animenewsnetwork.com/encyclopedia/ | スタッフ・職員クレジットの照合(監督・作画監督・原画)。 |
| **AniDB** | https://anidb.net/ | 作品・スタッフ・クリエイターのデータベース照合。 |

### 実写・撮影技法ソース(アニメでも実写でも当たる)

**アニメだけでなく、実在の映画撮影・照明・カラーグレードの技法ドキュメント**も能動的に検索して、prompt を**実在の技法用語**で埋める。

| ソース | URL | 用途 |
|---|---|---|
| **StudioBinder** | https://www.studiobinder.com/blog/ | 景別・カメラワーク・照明・カラーの実践技法を図解で確認。 |
| **American Cinematographer(ASC)** | https://theasc.com/ | 撮影監督の作例・レンズ/照明/露出の一次解説。 |
| **No Film School** | https://nofilmschool.com/ | 撮影・照明・グレードの実務記事。 |
| **ShotDeck** | https://shotdeck.com/ | 映画ショットのリファレンス(構図・色・光の参照)。 |
| **Wikipedia｜Cinematic techniques** | https://en.wikipedia.org/wiki/Cinematic_techniques | 技法用語の定義照合(焦点距離・被写界深度・運動)。 |

**実写の技法用語例(prompt を正向で埋める):** 焦点距離(24/35/50/85mm)・絞り(f/1.4〜f/8)・景別(LS/MS/CU/ECU)・カメラワーク(dolly/crane/steadicam/handheld/PAN/TU)・照明(三点照明・key/fill/rim・レンブラント・逆光)・色温度(3200K/5600K)・カラーグレード(LUT・ティール&オレンジ・フィルム調)・画面比・被写界深度。

**採取フロー:**

0. **まず能動的にウェブ検索する(必須)。** 課題のテーマ(アニメーター名・技法・作品・演出論点)で上記サイトと一般検索を当たり、開けるページを実際に取得する。記憶頼みで作者名や担当回を断定しない。
1. **先に開けるリンクを集める。** Sakugabooru でタグ検索 → 該当カットの URL とサムネ情報を控える。
2. **作者・職員を照合。** ANN / AniDB で監督・作画監督・原画担当を確認し、正確な作者名・作品名・話数を記録する。
3. **分析を読む。** Sakuga Blog / Sakuga Journal / Animétudes でそのカット/作家の技法解説を読み、**抽出すべき表演ロジック**(タメツメ・間・コマ・エフェクト設計)をメモする。
4. **素材パックにまとめる。** 参照リンク + サムネ + 九面格(キーポーズ抜き出し)/ 風格拆解(レイアウト・色・タイミングの分解)を一つの参考素材パックにする。
5. **技法語に翻訳して prompt へ。** 「誰々風」ではなく、抽出したタイミング・芝居・間・コマ・エフェクトを技法語で落とす。

**境界(重要):** 商業アニメは**学習参考**として分析・リンク列挙してよいが、**未授権の商業映像を一括ダウンロード・転載しない**。集めるのは「開ける参照リンク・サムネ情報・九面格/風格拆解の素材パック」まで。引用時は必ず作品名・話数・担当者・出典リンクを添える。

## A. 基礎原則(Disney / SIGGRAPH)

**学ぶ表演ロジック:** timing(タイミング)、staging(ステージング)、anticipation(予備動作)、secondary action(副次動作)、follow-through(フォロースルー)。アニメーションの普遍的な土台。

- timing:動きの長さと内部配分が、重さ・感情・コミカルさを決める。
- staging:一つの芝居を一つの読みやすい構図で見せる。
- anticipation:本番の前に観客を準備させる(跳ぶ前に沈む)。
- secondary action:主動作を支える副次の動き(歩きながら髪が揺れる、話しながら手が動く)。主を食わない範囲で生命感を足す。
- follow-through:主力が止まっても付随部(布・髪)が遅れて追う。

**prompt への落とし方:** 「予備動作→本番→フォロースルー」を時間ビートで明記し、副次動作(髪・布・小道具)を一つ添える。

**出典:**
- Disney Animation｜The Animation Process — https://www.disneyanimation.com/process/animation/
- SIGGRAPH｜Principles of Traditional Animation — https://education.siggraph.org/static/HyperGraph/animation/character_animation/principles/prin_trad_anim.htm

## B. ホールドと時間配分(AWN)

**学ぶ表演ロジック:** タイミングの「時間配分」と **hold(止め)** の使い方。動きそのものより、どこで止めて、どれだけ持たせるかが芝居を作る。

- hold(止め)で決定・気づき・情感の転換を読ませる。
- 全部を均一に動かさず、重要な瞬間に時間を割り当てる(時間配分=メリハリ)。
- 動きの前後の「持たせる尺」で印象が変わる。

**prompt への落とし方:** 「ここで○秒止める(hold)、その後一気に動く」を明示。間(ま)を尺で指定する。

**出典:** AWN Toon Institute｜Timing — https://www.awn.com/tooninstitute/lessonplan/timing.htm

## C. スメア & 部分ホールド(Canmom)

**学ぶ表演ロジック:** **smear frames(スメア=高速移行の引き伸ばし/多重残像)** と **partial hold(部分ホールド=体の一部だけ止め、他は動かす)**、およびリミテッド表現の省略美学(Zagreb School 的)。

- smear:速い移行の1〜2枚を残像・引き伸ばし・多重像で繋ぐ。直接見ると歪だが、再生で「速さ」になる。
- partial hold:顔は止め、手だけ動かす等、部分的に止めて注目を制御。
- リミテッドの省略:全部を動かさず、要る所だけ動かす(3コマ打ち/止め絵+口パク)。

**prompt への落とし方:** 高速の振りに「残像を伴うスメア的移行」、注目誘導に「体は止め、手と視線だけ動く部分ホールド」と技法語で指定。

**出典:**
- Canmom｜Smears — https://canmom.art/animation/smears
- Canmom｜The Zagreb School(リミテッド/省略) — https://canmom.art/films/animation-night/136-the-zagreb-school

## D. 芝居の機微 / 実写参考(Sakuga Journal・山田尚子の分析に学ぶ)

**学ぶ表演ロジック:** キャラの感情を **手・眼・姿勢の小さな芝居** で表す。身体の微細な動き(body micro-movements)と、実写的な芝居観察(real performance reference)。

- 大きな表情でなく、指先・足の運び・肩の落ち・視線の逃がしで情感を出す。
- 無意識のクセ(髪を触る、裾を握る)が「生きている人」を作る。
- セリフより「間」と所作で語らせる。実写の人の振る舞いを観察して写す。

**prompt への落とし方:** 「感情語」を禁じ、微表情と所作に変換する。例:「気まずさ」→「視線を逸らし、指で袖口をつまみ、半歩引く」。

**出典:** Sakuga Journal｜The Improvisational Beauty and Intentional Design of Naoko Yamada's *Kimi no Iro* — https://sakugajournal.wordpress.com/2025/04/19/the-improvisational-beauty-and-intentional-design-of-naoko-yamadas-kimi-no-iro/

## E. アクションのタイミング系譜(Animétudes / Wave Motion Cannon)

固有名は**系統の呼称**として技法理解のために使うが、生成 prompt には画風指定として書かない。抽出するのは各系統の**タイミング設計**。

### 金田系(Kanada-style)
**学ぶ表演ロジック:** キーポーズを強く決め、間を大胆に「抜く」スナップの効いたタイミング。誇張パース、幾何学的な面で構成するエフェクト(光・爆発・稲妻)。3コマ的なキレ。

**prompt への落とし方:** 「強いキーポーズ→抜き→一気のスナップ移行」「エフェクトは幾何学的な面で誇張」と書く。

### 松本憲生(流麗・重量のあるアクション)
**学ぶ表演ロジック:** 流れるように繋がりつつ、重みと衝撃を保つアクション。アクション内部の自然なタイミング変化。

**prompt への落とし方:** 「流麗に連続しつつ、接触に重みと反動を残す」「カット内で緩急を変化させる」。

### 大平晋也(リアリスト / 質感・変形)
**学ぶ表演ロジック:** リアル寄りの質感、ラフで密な線、独特の間隔・タイミング、物理の生々しさ。

**prompt への落とし方:** 「ラフで密な質感、生々しい変形と慣性」「均等割りを避けた不規則なタイミング」。

**出典:**
- Animétudes｜Artist Spotlight: Shinya Ohira — https://animetudes.com/2021/06/06/artist-spotlight-shinya-ohira/
- Animétudes｜The New Generation of the 1990s — https://animetudes.com/2021/06/04/the-new-generation-of-the-1990s/
- Animétudes｜The Kanada Style Now — https://animetudes.com/2021/07/09/the-kanada-style-now/

## 使い方まとめ

1. まず A・B(普遍原則とホールド)で芝居の土台を作る。
2. リミテッド/高速移行が要れば C(スメア・部分ホールド)。
3. 情感ドラマなら D(手・眼・姿勢の微細芝居・実写参考)。
4. アクションなら E(金田系=スナップと抜き / 松本=流麗+重量 / 大平=リアル質感)。
5. **いずれも「タイミングと芝居」を技法語で prompt に落とす。固有名で画風模倣を指定しない。**
