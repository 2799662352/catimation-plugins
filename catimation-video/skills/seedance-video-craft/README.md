# seedance-video-craft —— sd2-pe 的衍生模块

> 本 skill 是 **`sd2-pe`(视频与提示词的总兜底/起点)的衍生模块**,坐在 sd2-pe 产出的工程化提示词之上,不替代它。

## 分工

| 层 | skill | 负责 |
|---|---|---|
| 骨架(先) | **sd2-pe** | 把需求/草稿/多模态 JSON 重写为工程化提示词:八大要素、任务分类、素材映射、路径 A/B 三段论、强制兜底包 |
| 衍生(后) | **seedance-video-craft**(本 skill) | 模型对齐 + 出片编排 + 爆款体检:Seedance 2.0 满血版真实能力(4–15s/720p/最多 9 图·3 视频·3 音频参考/原生音频口型)、全能参考配方与配额分配、广告/短视频模式分类、爆款体检自评 |

## 用法

1. 先用 sd2-pe 跑出工程化提示词(`/sd2` 或 `/gen-video` 第 1 步)。
2. 用本 skill 校准:该提示词是否吃满 Seedance 2.0 能力?全能参考配额怎么分(角色图/场景图/运镜视频/音频)?过不过爆款体检?
3. 把校准后的结果交 `catimation-video` 的 `generate_video` 出片。

> 即:**sd2-pe 决定"提示词怎么写",seedance-video-craft 决定"怎么把模型能力吃满 + 出爆款"。**
